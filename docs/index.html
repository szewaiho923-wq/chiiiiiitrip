Day1 — 交通：深圳 → 張家界

長途高鐵與轉車，建議預留充足休息時間。

袁家界（哈利路亞山 / 乾坤柱）

袁家界屬於武陵源核心景區，砂岩峰林地貌在 3.8 億年前由地殼抬升與流水侵蝕所形成。山峰奇形怪狀，其中著名的「乾坤柱」（哈利路亞山）為好萊塢電影《阿凡達》中懸浮山的原型；「天下第一橋」為兩座山峰之間天然形成的巨大石橋，景觀震撼。

人流：山頂核心觀景平台與百龍天梯排隊人潮多。建議穿舒適鞋並安排充裕休息。

乾坤柱百龍天梯（無長者優惠）

天子山與金鞭溪

天子山被譽為「峰林之王」，有御筆峰等著名景觀；金鞭溪兩岸奇峰對峙、水質清澈，溪谷平路徒步路線被稱為「山水畫廊」。金鞭溪徒步時間約 2.5 小時，對長者體能是考驗。

小提示：金鞭溪段相對幽靜，但天子山觀景台、索道站仍有人流集中，建議早點出發避開尖峰時間。

天門山（天門洞、玻璃棧道）

天門山主峰高 1518.6 米，最標誌為天門洞——一個巨大的穿山溶洞，傳說在公元 263 年山壁忽然洞開。山頂的環山棧道與玻璃棧道沿懸崖修築，視野壯觀但對部分遊客具心理挑戰。

注意：索道與天門洞前廣場常有長時間排隊；若同行長者需評估心理接受度與行動能力。

嶽麓書院

北宋書院，環境幽靜，步行輕鬆。

湖南省博物館（馬王堆）

湖南省博物館以收藏馬王堆漢墓的珍貴文物而聞名，出土文物完整且保存良好，能深入了解西漢時期的生活與禮儀。參觀多為室內展館，可能需要較長時間站立與行走。

建議：安排坐休息與緩慢行程節奏，避免長時間連續站立。

橘子洲頭

橘子洲頭因毛澤東早年在此揮毫並寫下《沁園春·長沙》而成名，河岸綠地與雕塑區常聚集遊客與市民，景色富詩意。

建議：若同行長者不便長時間行走，考慮搭乘觀光車或分段漫步。

黃鶴樓

黃鶴樓是江南三大名樓之一，始建於三國時期，因唐代詩人崔顥的詩句而聲名遠播。現存建築為 1980 年代重建，保留古典風格並為欣賞長江景致的制高點。

注意：登樓需爬樓梯；景區有休息平台與電梯可供較不便旅客使用。

湖北省博物館（曾侯乙編鐘）

湖北省博物館收藏大量古楚文化遺物，其中曾侯乙編鐘為重要亮點，為世界上保存最完整的大型青銅樂器之一。博物館展品豐富，適合喜歡歷史文化的旅客。

建議：博物館內人流可能集中，東湖可以選擇環湖觀光車以減少步行負擔。

滕王閣

滕王閣最早建於唐朝永徽四年（公元 653 年），因王勃《滕王閣序》而名垂千古。現存建築依宋代樣式重建，位於贛江邊，風景與文化氣息濃厚。

建議：安排緩步登樓並在觀景平台休息，注意人流高峰時段。

陶溪川文創街區

陶溪川由 1950 年代的宇宙瓷廠改造而成，現為文創聚落，匯集年輕陶藝家與手作市集。每週末夜市活力十足，是觀賞與購買手作的好去處。

必買伴手禮

景德鎮古窯民俗博覽區

古窯區保留清代鎮窯、明代葫蘆窯等歷代窯爐，展示完整的製瓷流程，從揉泥、拉坯到燒製都能近距離觀察，是體驗「活著的陶瓷歷史」的最佳場所。

江西省博物館

江西省博物館收藏大量地方文物，呈現江西在歷史長河中的文化與地位。南昌在近代史上有特殊意義，如八一南昌起義的歷史紀念，博物館適合安排半日深入參觀。
